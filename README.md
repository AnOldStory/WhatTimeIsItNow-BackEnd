What Time Is It Now - BankEnd V1.0.0
===============================

Description
-----------

> 이 저장소는 오픈소스를 사용하여 디지털 콘텐츠를 공유하는 서비스를 제작 할 수 있는 웹 페이지의 서버입니다.


Installation
------------

Node.js + express + mysql + sequlize를 사용하여 제작 되어진 웹 페이지의 입니다.

~~~
$ npm install

$ npm start
~~~
